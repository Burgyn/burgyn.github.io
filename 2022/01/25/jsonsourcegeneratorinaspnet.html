My troubles with the new JSON source generator in the ASP.NET API

Jan 25, 2022 | Milan Martiniak | C#, Generators, ASP.NET, .NET

New in .NET 6 is the C# JSON generator. In some cases, benchmarks promised up to ~40% increase in serialization performance (For example here.).

I wanted to test the impact this will have on real use cases in ASP.NET API projects. It should be easy to use:

{
