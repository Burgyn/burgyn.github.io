When I find an interesting article, I don't read it immediately. Instead, I bookmark it using raindrop.io - my go-to bookmarking system for everything.
Articles I want to read go into a "Blogs" folder, which I've set up as a public RSS feed. This RSS feed is connected to Instapaper - a reader that creates a clean copy of the article, optimizes it for mobile reading, and most importantly: preserves scroll position. Whenever I return to an article, I'm exactly where I left off.
